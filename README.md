# QA Automation Challenge - ColmeIA

This repository contains an automated E2E test suite for the ColmeIA platform using **Cypress**

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npx cypress run` (Headless) or `npx cypress open` (Interactive)

You can also check the Actions page to see CI test results.

## Identified Bugs
The complete bug report can be found in [BUG_REPORTS.md](./BUG_REPORTS.md).
